# UptimeKuma Agent

[English](README.md) | [简体中文](README_zh-CN.md)

一个用于 UptimeKuma 的代理客户端，用于监控 shell 命令、systemd 服务或其他服务。

[警告] 此仓库仍在开发中

### 未来功能

- Systemd 服务监控
- 自定义 bash 命令监控
- HTTP 请求监控
- 多线程监控机制
- 完善的日志记录
- 易于使用
- 一键云端控制

### 安装

目前计划使用 systemd 服务（将 Python 脚本部署为服务）

### 使用方法

只需填写配置文件，然后通过 `python3 main.py` 运行程序
